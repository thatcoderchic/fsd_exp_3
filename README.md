# NEXUS SPA

A fast, beautifully designed Single Page Application built with **React**, **Vite**, and **Material UI**. This project features a custom dark mode theme with neon accents, smooth client-side routing, and a responsive layout.

## Features
- ⚡️ **Vite & React**: Blazing fast hot-module replacement and optimized builds.
- 🎨 **Material UI**: Robust component library fully themed for a premium look.
- 🧭 **React Router**: Seamless client-side navigation between pages.
- 📱 **Responsive**: Fully responsive and mobile-friendly design.
- 💅 **Custom Aesthetics**: Neon gradients, glassmorphism overlays, and micro-animations.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Development Server**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:5173/`.

3. **Build for Production**
   ```bash
   npm run build
   ```
   The production-ready files will be generated in the `dist` directory.

## Project Structure
- `/src/components`: Reusable UI components (like Layout, Navbar).
- `/src/pages`: Main view routes components (Home, Dashboard).
- `/src/theme.ts`: Custom Material UI theme configuration.
- `/src/App.tsx`: Routing and ThemeProvider setup.
